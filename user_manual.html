スマホへのインストール（推奨）

全員

ホーム画面に追加しておくと、アプリのようにすぐ起動できます。

iPhone（Safari）の場合

  1. 1Safari でサイトURLを開く（Chromeでは不可）
  2. 2画面下の 共有ボタン（□↑） をタップ
  3. 3「ホーム画面に追加」 をタップ
  4. 4右上の 「追加」 をタップして完了 ✅

Android（Chrome）の場合

  1. 1Chrome でサイトURLを開く
  2. 2右上の メニュー（⋮） をタップ
  3. 3「ホーム画面に追加」 をタップ
  4. 4「追加」 をタップして完了 ✅
💡インストール後はホーム画面のアイコンをタップするだけで起動できます。アプリのように使えます。

支出を登録する

一般会員

立替払いや経費を申請するときに使います。領収書を撮影するとAIが自動で金額・日付・支払先を読み取ります。

登録の手順

  1. 1ホーム画面の 「📷 支出を登録」 をタップ
  2. 2領収書を撮影（またはギャラリーから選択）
    → AIが自動で内容を読み取ります（数秒かかります）
  3. 3読み取り内容を確認・修正する
  4. 4「支出を登録する」 ボタンをタップ
  5. 5「登録しました！」と表示されたら完了 ✅

入力項目の説明

項目説明必須
提出者自分の名前（プルダウンから選択）✅
日付支払った日付（AIが自動入力）✅
事業区分費用の種類（連合関連・交際費など）✅
金額支払金額（AIが自動入力）✅
数量例：30枚、5個、1パックなど任意
但し書き例：御祭礼費として任意
支払先例：○○酒店、○○スーパー任意
備考その他メモ任意
💡複数の領収書をまとめて選択すると、1枚ずつ順番に登録できます。
💡領収書なしで登録することもできます。写真なしで各項目を手入力してください。

収入を登録する

一般会員

奉納金・会費・補助金など、収入があった場合に記録します。

  1. 1ホーム画面の 「💴 収入を登録」 をタップ
  2. 2各項目を入力（支出と同じ操作）
  3. 3「収入を登録する」 ボタンをタップ

収入の事業区分の種類

区分例
年会費会員からの年会費
町会補助金町内会からの補助
祭礼奉納御祭りへの奉納金
直会参加費直会・宴会の参加費
個人奉納個人からの奉納
預金利息銀行口座の利息
その他上記以外

提出履歴を確認する

一般会員

自分が過去に提出した内容や、精算状況を確認できます。

  1. 1画面下の 「📋 履歴」 タブをタップ
  2. 2プルダウンから 自分の名前 を選択
  3. 3一覧が表示されます
  4. 4各項目をタップすると詳細を確認できます

バッジの意味

バッジ意味

未精算の管理

管理者

精算済みにする

  1. 1精算したい項目の チェックボックス をタップ（複数選択OK）
  2. 2右上の 「精算済みにする」 ボタンをタップ
  3. 3「〇件を精算済にしました」と表示されれば完了 ✅
💡精算したデータは「精算済みタブ」に移動します。履歴からも確認できます。

データを修正する

  1. 1修正したい行の右端 ✏️ ボタン をタップ
  2. 2日付・金額・区分などを編集
  3. 3「保存」 をタップ

データを削除する

  1. 1削除したい行の右端 🗑️ ボタン をタップ
  2. 2確認ダイアログで 「OK」 をタップ
🚫削除したデータは元に戻せません。誤って削除しないよう注意してください。
✅

精算済みデータの管理

管理者

精算が完了したデータを後から確認・修正・削除できます。

年度を絞り込む

左上のプルダウンで年を選択（2026・2025・…）すると、その年のデータのみ表示されます。自動で更新されます。

データを修正する

  1. 1対象行の ✏️ ボタン をタップ
  2. 2日付・金額・位分・但し書きなどを編集
  3. 3「保存」 をタップ

未精算に戻す（誤精算の修正）

誤って精算してしまった場合に、未精算の状態に戻せます。

  1. 1対象データの チェックボックス をタップ
  2. 2「↩ 未精算に戻す」 ボタンをタップ
  3. 3確認ダイアログで 「OK」 → 未精算タブに戻ります

データを完全削除する

  1. 1対象行の 🗑️ ボタン をタップ
  2. 2確認後に完全削除されます
🚫削除したデータは元に戻せません。必要な場合は事前にスプレッドシートでバックアップしてください。

会計レポートの確認・PDF出力

管理者

収支レポートを確認する

  1. 1「📊 会計」 タブをタップ
  2. 2年度プルダウンで確認したい年を選択 → 自動で更新

表示内容の説明

表示項目説明
前年度繰越金昨年から引き継いだ残高
収入合計当年度の全収入合計
支出合計当年度の全支出合計
次年度繰越金繰越金 ＋ 収入 － 支出
収入内訳事業区分ごとの収入金額
支出内訳事業区分ごとの支出金額（予算比ゲージ付き）
💡会計レポートは精算済み・未精算に関わらず登録された全データを集計します。

会計報告書PDFを出力する

  1. 1レポート画面下の 「📄 会計報告書を生成 (PDF)」 をタップ
  2. 2数秒待つと自動でダウンロードが始まります
  3. 3会計報告書_2026.pdf というファイルで保存されます
⚙️

マスターデータの管理

管理者

提出者リストを編集する

  1. 1「⚙️ マスター」 タブ → 「提出者」 を選択
  2. 2入力欄に名前を入力 → 「追加」 ボタン
  3. 3▲▼ ボタンで順番を変更、✕ ボタンで削除
  4. 4「変更を保存」 をタップして確定

前年度繰越金を更新する

  1. 1マスタータブ → 「前年度繰越金」 を選択
  2. 2金額を入力（例：1,211,366）
  3. 3「変更を保存」 をタップ
💡ホーム画面に表示される繰越金もここを変更することで更新されます。

よくある質問

全員
提出者の名前が選択肢にない
管理者に連絡して「マスター管理」から追加してもらってください。
登録したのに履歴に表示されない
履歴タブで名前を選び直すと更新されます。またはアプリを完全に終了（タスクキル）して再起動してください。
OCRで読み取った金額が違う
フォームの金額欄を手動で修正してから登録できます。AIの読み取りミスは手動で直せます。
精算済みにしたのに会計レポートに反映されない
会計レポートは精算済み・未精算に関わらず全データを集計します。年度選択が正しいか確認してください。
ログインしても管理者画面が開かない
管理者キーが正しいか確認してください。それでも解決しない場合はアプリを再起動してください。
画面が古いデザインのまま更新されない
アプリを完全に終了（タスクキル）してから再起動してください。iPhoneの場合は画面下から上にスワイプしてアプリを閉じてください。
